# Screenshots

Drop your screenshots here. These are referenced in the root README.md.

## Required files

| Filename | What to capture |
|----------|----------------|
| `landing.png` | Landing page hero with CTA buttons |
| `discovery-list.png` | Trip discovery in list view with filter bar visible |
| `discovery-map.png` | Map view with trip markers and a popup open |
| `trip-details.png` | Trip details page showing participants + chat |
| `group-chat.png` | Group chat with a few messages |
| `profile.png` | A filled-out profile page |
| `mobile.png` | Any screen at mobile width (375px) — use browser DevTools |

## Tips

- Use browser zoom at 80% to fit more content in the screenshot
- Use a real account with some trip data for the best looking screenshots
- Recommended size: 1280×800px for desktop, 375×812px for mobile
- Save as PNG for crisp text
